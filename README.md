# 🌦️ Weather App with Search History

This is a Full Stack Weather Application that shows real-time weather information using OpenWeatherMap API and stores search history in MongoDB.

---

## 📌 Features

• Search weather by city name  
• Shows:

- Temperature
- Humidity
- Wind Speed
- Weather condition

• Stores search history in MongoDB  
• Fast and responsive UI  

---

## 🛠️ Technologies Used

Frontend:

- HTML
- CSS
- JavaScript

Backend:

- Node.js
- Express.js

Database:

- MongoDB
- Mongoose

API:

- OpenWeatherMap API
